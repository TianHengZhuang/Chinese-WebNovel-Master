# Quick Start

## Introduction

Chinese WebNovel Master is a multi-agent AI writing system for Chinese web fiction.

It transforms a simple idea into a complete commercial novel package.

Workflow:

User Idea

↓

Planner Agent

↓

Architect Agent

↓

Writer Agent

↓

Editor Agent

↓

Publisher Agent

---

## Step 1

Open Planner Agent.

Input:

```text
Write a cultivation novel for Qidian.

Requirements:

- Male audience
- Fast progression
- Sect development
- Long-term power growth
```

Output:

* Genre analysis
* Audience analysis
* Commercial score
* Core fantasy

---

## Step 2

Pass Planner output to Architect Agent.

Architect Agent generates:

* World setting
* Power system
* Character design
* Long-term plot

---

## Step 3

Pass Architecture to Writer Agent.

Writer Agent generates:

* Opening chapters
* Scene structure
* Dialogue
* Hooks

---

## Step 4

Pass chapters to Editor Agent.

Editor Agent reviews:

* Logic
* Consistency
* Pacing
* Reader retention

---

## Step 5

Pass approved manuscript to Publisher Agent.

Publisher Agent generates:

* Titles
* Synopsis
* Tags
* Marketing copy
